# Judge Demo Script

## 2-3 Minute Flow

1. **Open the landing page**
   - "Xentinel is an AI-powered DeFi Risk Co-Pilot + Smart Money Sentinel."
   - "The user journey is simple: paste wallet, analyze portfolio, stress test, compare smart money, detect panic, ask the AI."

2. **Launch the dashboard**
   - Point out the command-center layout and module navigation.

3. **Portfolio Guardian**
   - Use an example wallet.
   - Show overall risk, rating, intrinsic risk, systemic risk, and highest-risk position.

4. **Stress Testing Engine**
   - Show exit liquidity and downside timing.
   - Explain that risk is not just score; it is also how fast a user can exit.

5. **Smart Money Sentinel**
   - Show the preloaded watchlist.
   - Show "your portfolio risk vs smart-money average."

6. **Panic / Outflow Detector**
   - Show panic meter and early-warning events.

7. **Contagion Radar**
   - Show how exposure travels from asset to protocol to wallet.

8. **AI Chat Co-Pilot**
   - Ask: "Which position worries you most?"
   - Point out that the response is grounded in portfolio, stress, panic, and dependency context.

9. **Beautiful Outputs**
   - Show the wallet risk brief preview and alert cards.
   - Explain that downloadable reports appear when the live report tool returns an artifact URL.

## Close

"Xentinel is not another balance dashboard. It is a risk workflow that helps users understand what is exposed, what could break next, and what deserves attention before panic becomes obvious."
