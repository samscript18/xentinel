# Why Xerberus

Xentinel is uniquely enabled by Xerberus because DeFi risk requires more than balances.

Xerberus provides the risk primitives Xentinel needs:

- Ratings
- Intrinsic risk
- Systemic risk
- Wallet and entity analysis
- Exit-liquidity and portfolio ladder tooling
- Stress testing
- Dependency and contagion context
- Report and watch tooling when enabled

Xentinel turns those primitives into a user-facing product:

- Portfolio Guardian explains current risk.
- Stress Testing Engine shows what happens when exits matter.
- Smart Money Sentinel compares user posture against watched wallets.
- Panic / Outflow Detector connects risk score, exit pressure, and smart-money changes.
- AI Chat Co-Pilot translates risk context into plain English.
- Beautiful Outputs makes results shareable.

The result is a practical early-warning system rather than a static dashboard.
